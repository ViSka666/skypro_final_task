Основные Docker команды

Docker (Докер) — это программное обеспечение с открытым исходным кодом для разработки, тестирования, доставки и запуска веб-приложений в средах с поддержкой контейнеризации.
  • docker run - Запуск контейнера.
  • docker ps - Просмотр запущенных контейнеров.
  • docker stop / docker start - Управление состоянием контейнера
  • docker exec - Вход в запущенный контейнер
  • docker build - Сборка образа из Dockerfile
    • _____________________________________________
  • docker --help - Справка по всем командам.
  • docker command --help - Справка по конкретной команде.
